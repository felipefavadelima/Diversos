Física do Modelo Matemático: O equilíbrio térmico se dá quando a potência gerada pelo Efeito Joule iguala a perda por convecção (Lei de King).
Pgerada = I² · Rw
Pperdida = (A + B√Q) · (Tw - Tf)
A resistência do fio de tungstênio varia com a temperatura: Rw = R₀ [1 + α(Tw - Tf)].
O simulador resolve este sistema algebricamente a cada frame. O resultado é a curva não linear vista no gráfico. Ventiladores modernos usam circuitos de Temperatura Constante (CTA) acoplados a microprocessadores para linearizar essa leitura.
